Multiple Calendars & Getting Started

Multiple Calendars per Year: You can create multiple calendars for the same year by giving them different names (e.g., "Family", "Work", "School"). Each calendar can have different events from different master event groups.

After creating your calendar, you'll be able to:

  • Apply master events from your event groups
  • Upload images with MMDD_eventname format (e.g., 0314_birthday.jpg for March 14th birthday)
  • Import header documents for calendar decoration
  • Generate PDF calendars in various formats
  • Combine headers with calendar pages for professional layouts

Master Events: Use the "Events" and "Groups" links in the navigation to manage your recurring events that can be reused across multiple calendars.
